# RLTumor

Reinforcement learning framework for microswimmer tumor navigation.

## Features
- PPO-based navigation
- Obstacle avoidance
- Lidar sensing
- Hybrid wall-following controller
- Randomized tumor environments

## Structure

- envs/ → environments
- training/ → PPO training scripts
- analysis/ → evaluation and plots
- notebooks/ → experimental development
- results/ → generated outputs
- models/ → trained models

## Run

python training/train.py
